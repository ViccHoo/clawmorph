# ClawMorph v0.1.0

First public MVP release of ClawMorph.

## Included
- list available role packs
- preview workspace changes before mutation
- apply role packs safely
- create snapshots before mutation
- rollback the latest applied role pack
- demo fixture for local testing

## Built-in role packs
- researcher
- designer
- lawyer
- product-manager
- founder

## Notes
This release focuses on a lightweight CLI workflow for OpenClaw agent workspaces. It does not include a web UI, plugin packaging, or a role marketplace yet.
